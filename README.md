# SimpleGeo — тренажёр ОГЭ по географии

Статический сайт на React + Vite. Банк заданий ФИПИ (1360 шт): одиночный выбор, выбор нескольких, короткий ответ. Фильтр по разделам, подсчёт результата, картинки грузятся с сервера ФИПИ.

---

## Локальный запуск

Нужен установленный Node.js (версия 18+). Проверить: `node --version`.

```bash
npm install      # поставить зависимости (один раз)
npm run dev      # запустить локально → открой http://localhost:5173
npm run build    # собрать продакшн-версию в папку dist/
npm run preview  # посмотреть собранную версию локально
```

Всё, что видит пользователь, собирается в папку **`dist/`** — именно её содержимое заливается на хостинг.

## Деплой на simplegeo.ru (Selectel, объектное хранилище S3)

Зарубежные хостинги (Vercel/Netlify/Cloudflare) в РФ нестабильны — используем Selectel, там же где домен.

### Шаг 1. Создать контейнер
1. Панель Selectel → Объектное хранилище → создать **публичный** контейнер (тип: для статических сайтов).
2. В настройках контейнера указать индексную страницу `index.html`.

### Шаг 2. Залить сайт
1. Локально: `npm run build`.
2. Загрузить **всё содержимое папки `dist/`** в контейнер (через веб-панель перетаскиванием, либо через S3-клиент — см. ниже).
3. Сайт откроется по служебному домену контейнера — проверь, что работает.

### Шаг 3. Привязать домен
1. В контейнере → вкладка «Домены» → добавить `simplegeo.ru` (и/или `www.simplegeo.ru`).
2. Selectel → DNS-хостинг для `simplegeo.ru` → добавить запись по инструкции, которую покажет панель при привязке (обычно CNAME на домен контейнера).
3. Включить HTTPS для домена (в панели Selectel — выпуск сертификата).
4. Подождать обновления DNS (от 10 минут до нескольких часов).

### Автозаливка через S3-клиент (удобно при частых обновлениях)
```bash
# поставить aws-cli, затем настроить профиль с ключами Selectel S3 (Доступ → S3-ключи)
aws s3 sync ./dist s3://ИМЯ_КОНТЕЙНЕРА \
  --endpoint-url https://s3.storage.selcloud.ru \
  --delete
```
Точный endpoint и ключи бери в панели Selectel. Ключи держи только у себя, в код не коммить.
